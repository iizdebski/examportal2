# Spring Boot E-Commerce

## ***Technologies used:***

* **Java 14**
* **Spring Boot**
* **Angular 12**
* **Spring Security (Sign up / Login)**
* **JWT Token**
* **Admin Dashboard**
* **Timer Implementation in Quiz**
* **MySQL**
* **Material UI**
* **Maven**
